=== Version 1 - [13-03-2026] ===

Initial version.

Consists of a fork from Version 0.61 of the game program "ReMined"
by Nelson Elhage

This was obtained from the program’s project repository hosted at
the web-based project hosting service provider SourceForge.

Notes, observations and/or changes follow:

    * Added a "Front Matter" component to the program from which users
      can set the parameters that they wish ReMined to use and also
      find information such as how to play ReMined and credits.
    
    * Renamed the main() function of ReMined to original_main()
      in-order to allow the program to begin in the new Front Matter
      using a main() function there.
    
    * Moved the initilisation of the SDL API from the original
      ReMined code to the new Front Matter code.

    * Moved code setting default difficulty settings from original
      ReMined code to the new Front Matter code.

    * Included a section at the beginning of each source code file
      which contains information about what program they are a
      component of.

*** Version 0.61 - 2006-02-03 ***

* Moved everything over to automake and autoconf, so you can now
  actually install, rather than running out of the build directory

*** Version 0.6 - 2005-08-23 ***

* Improved performance: No longer hogs 100% CPU even when idle, and
  stays at 0% during expert play on my 1.7Ghz Pentium

* Added command-line arguments to specify the board size, as well as
  presets for beginner, intermediate, and expert, and an option to use
  or not use the '?' ``mark'', as well as a usage message

* F2 now starts a new game, like on Windows

* The timer now properly stops at 999

* The cheat now continues to toggle after game over

* The cheat now requires right shift at the end of the combination,
  like on Windows

* Palettized all the bitmaps, significantly reducing size on disk

* Some minor code cleanup and beautification, including an attack of
  80-column OCD.

* `make clean' now actually deletes the .o

*** Version 0.56 - 2005-08-07 ***

* Middle-click now functions like a both-click. Thanks to Simon Biber
  pointing out it should.


*** Version 0.55 - 2005-05-26 ***

* Added ctrl-click (CMD-click on OS X) to function like a both-click

* Added the cheat Easter egg, almost the same as on Windows, but with
  two distinctions:
  - It uses (0,0) of the game board, not the monitor
  - It doesn't require the `shift' at the end of the combo

*** Version 0.50 - 2005-05-09 ***

* Initial release
